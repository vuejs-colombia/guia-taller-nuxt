## Crea tu primer proyecto Nuxt


``` sh
npx create-nuxt-app <nombre-de-proyecto>

# o con yarn

yarn create nuxt-app <nombre-de-proyecto>
```

Escribir instrucciones posteriores

``` sh
cd <nombre-de-proyecto>
npm run dev
```
